一.redis配置说明：
        1.打开项目node-shoots\resource\redis_config.js
        2.修改商品的redis路径  请找到文件 53 行位置处  对new ProductConfig("192.168.10.66",6380,"redis23")进行替换，修改相应的 Ip  端口  密码
        3.修改项目session信息保存路径  请找到文件 63 行， 修改对应的   host属性即ip信息  port端口信息 ，  也可以设置session的 失效时间 ttl属性 单位以s计
                                    - - -TIP- - -请注意 session 的配置有两个 一个是只读一个是可读写 两个都要配置 而且是一致的

        5.修改 同步订单服务的 配置  请找文件的 112 行， 替换相应的ip 与 服务的端口号
        6.修改 删除redis信息的 配置  请找文件的 117 行， 替换相应的ip 与 服务的端口号
